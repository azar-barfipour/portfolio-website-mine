my, myself and I

The first time in my life that I could choose my major, was at the end of high school when I chose Software Engineering. I was really curious about computers and I learned many various aspects of this field during my four years at university. In the last year of study, there was a workshop about web development, we created a simple website. It was my first time creating a website and it was very interesting when I saw the very first Hello World on my screen.

After graduation, I joined a big company and I worked there for more than four years. It was a great opportunity to collaborate with more than 40 teammates and handle more than 100 clients, debugging, testing, and developing forms by using HTML, CSS, JavaScript, and SQL Server. However, I gained huge experiences there but still, had a passion for creating websites.

Then, I moved to Canada to achieve my old goal which is frontend development. I started self-studying web development and joined CICCC college and gained many valuable and updated knowledge of web development such as SASS, React.js, Node.js and Ecma6.

Then, I started to work as a frontend developer volunteer at Freelivenet company until now, Gaining many helpful experiences working on a Angular app using git and Bitbucket

Meetup App

This is a react app for finding events to join and having fun. Users can search for their favorite events and book them. Booked events would be shown on each user's calendar. Also, if the user decided to not attend the event can easily delete the event from the calendar. Moreover, users can add their event, so others can see or choose it. (personal project)

Technical specifications: React, HTML, CSS, firebase

Weather-Location App

This is an api using react app for checking the weather by location. Users can see their location weather by allowing the access to their location. Moreover, users can check any places in the world by finding it on the map and click on it. (personal project)

Technical specifications: React, HTML, CSS, using two apis : google map api, open weather api

E-Commerce App

This is an e-commerce react app, using redux. Users after login can see the handy crafts products and they are able to add them to the card. Moreover, on the card, products will insert and delete. (personal project)

Technical specifications: React, HTML, CSS, using google authentication and firebase

Portfolio Website

This portfolio website. It is a website without using frameworks. It was a great experience creating a website from scratch without using any frameworks such as React or Angular. (personal project)

Technical specifications: HTML, CSS, BootStrap, JavaScript, JQuery
